Política de Privacidade

Aplicativo: Birdsong Studio
Última atualização: abril de 2026

1. Coleta de dados

O Birdsong Studio não coleta, armazena nem transmite nenhum dado pessoal para servidores externos. O aplicativo funciona inteiramente de forma offline.

2. Arquivos de áudio

Os arquivos de áudio que você carrega no aplicativo são armazenados exclusivamente no armazenamento local do seu dispositivo (IndexedDB). Esses arquivos nunca saem do seu dispositivo e não são compartilhados com terceiros.

3. Presets e configurações

As configurações de presets são salvas localmente no dispositivo (armazenamento interno do aplicativo). Nenhuma informação de configuração é enviada para a internet.

4. Permissões utilizadas

  • Leitura de arquivos de mídia: necessária para que você possa selecionar arquivos de áudio do seu dispositivo.
  • Serviço em primeiro plano: permite que o áudio continue tocando com a tela bloqueada.
  • Wake lock: mantém o processador ativo durante a reprodução em segundo plano para evitar interrupções.
  • Notificações: exibe o status de reprodução na barra de notificações.

Nenhuma dessas permissões é usada para coletar dados pessoais.

5. Anúncios e rastreamento

O Birdsong Studio não exibe anúncios e não utiliza nenhuma biblioteca de rastreamento ou análise de comportamento.

6. Crianças

Este aplicativo não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de crianças.

7. Alterações nesta política

Caso esta política seja atualizada, a nova versão será publicada nesta mesma página com a data de atualização revisada.
